Ejercicio 8

A partir de las siguientes reglas CSS:

body { font-family: Arial; color: black; } h1 { font-family: Verdana; }
p { color: red; }
Y del siguiente código HTML:
<body> <h1>Titular de la página</h1> <p>Un párrafo de texto.</p> </body>

Titular de la página

Un párrafo de texto.

¿Se aplica la herencia a algún elemento? Si es así, ¿Qué propiedades se heredan?
Si porque en la etiqueta body los elementos dentro de el, hereda todas la propiedades. Pero puedes cambiar cualquier propiedad con solo poner una etiqueta y asignándole un color, un tamaño, ect.
