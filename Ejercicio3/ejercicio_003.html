Explica la salida por pantalla.
Para que se aplique la regla del css los elementos "a" tiene que ser hijos directos de "p".
Si ponemos un "span" entre el "p" y "a" ejemplo<p><span><a></a></span></p> no se cumple la regla del css por "a" no se hijo de "p".
¿Cuáles son las diferencias con el selector de la pregunta anterior?
En la pregunta anterior no tiene una regla que tiene que seguir obligatoriamente, solo tiene seguir que si hay una "a" detro de "p" (aunque hay algun "span" en el medio )
