Ejercicio 6

Elimina las reglas anteriores. Aplícale las siguiente y explica para qué sirven.

a:link {color: red; text-decoration: none;}
a:visited {color: blue; text-decoration: none;}
a:hover {color: red; text-decoration: underline;}

Google

Bing

Explica para qué sirve cada una de ellas.
  1. En la primera regla se cumple en la etiqueta "a" poniendolo el texto de color rojo y no tiene ninguna decoracion (por defecto un enlace esta subrayado).
  2. En la segunda regla se cumple cuando has visitado el enlace poniendolo el texto de color azul y no tiene ninguna decoracion (por defecto un enlace esta subrayado).
  3. En la tercera regla se cumple cuando el cursor esta encima del enlace poniendo el texto de color rojo y como en la propiedad "text-decoration: underline" nos pondra un subrayado .
