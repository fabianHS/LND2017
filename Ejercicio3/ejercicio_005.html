Ejercicio 5

Crea una página HTML que tenga dos enlaces a dos páginas externas. Aplícale las siguientes reglas:

a:link {color: red;}
a:visited {color: blue;}
a:hover {color: fuchsia;}

Google

Bing

Explica para qué sirve cada una de ellas.
  1. En la primera regla se cumple en la etiqueta "a" poniendolo el texto de color rojo.
  2. En la segunda regla se cumple cuando has visitado el enlace poniendolo el texto de color azul .
  3. En la tercera regla se cumple cuando el cursor esta encima del enlace poniendo el texto de color fucsia.
