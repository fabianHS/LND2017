Ejercicio 2. Estructura de los canales de contenidos

  1. Indica para qué se usan los siguientes elementos hijos del elemento en un documento RSS:

    2. <category>

    Hace posible que los agregadores RSS agrupen sitios según la categoría.

    <copyright>

    Notifica sobre material protegido por derechos de autor.

    <image>

    Permite que se muestre una imagen cuando los agregadores presentan un feed.

    <language>

    Se usa para especificar el idioma utilizado para escribir su documento.

  2. Indica para qué se usan los siguientes elementos hijos del elemento en un documento RSS:

    3. <author>

    Se usa para especificar la dirección de correo electrónico del autor de un artículo.

    <comments>

    permite que un elemento se vincule con comentarios sobre ese elemento.

    <enclosure>

    permite que se incluya un archivo multimedia con un elemento.
